# 项目说明
客户端的自动更新程序，目前是 windows 版本的，因为是 lazarus 开发的，所以大家其实很容易改写为其他平台。
下载更新的内容都是可配置的，所以实际上绝大多数项目直接修改配置文件使用就可以了，根据连重新编译都不需要。目前我们的 xmppmini 和 eEmail 的自动更新程序就是同一个 exe ，只是配置不同而已。
